Redis-Set操作命令

命令参数说明

  1. 带有操作元素个数参数(count)
    • count是整数且小于元素的个数，返回含有count个不同的元素的数组，
    • count是个整数且大于集合中元素的个数时，返回整个集合的所有元素，
    • count是负数，则会返回一个包含count的绝对值的个数元素的数组，如果count的绝对值大于元素的个数，则返回的结果集里会出现一个元素出现多次的情况.

设置和获取操作

sadd、smembers
srandmember
sscan

判断操作

scard
sismember

集合操作

sdiff、sdiffstore
sinter、sinterstore
sunion、sunionstore
smove

删除操作

spop
srem
