Posted on 2022-06-24 Edited on 2023-03-28 In Redis

Redis-List操作命令

命令参数统一说明:

  1. 索引（index）：

    • 0表示第一个，1表示第二个，以此往后推
    • -1表示列表的最后一个，-2表示倒数第二个，以此往前推

  2. 阻塞时的超时时间（timeout）

    表示的是一个指定阻塞的最大秒数的整型值，Redis版本>=6.0，timeout 支持双精度浮点。

    当timeout为0是表示永久阻塞

设置和获取操作

lpush、rpush、lrange
lpushx、rpushx
lset、lindex
linsert
ltrim

判断操作

llen

删除操作

lrem
lpop、rpop
blpop、brpop
rpoplpush、brpoplpush

从 Redis 6.2.0起

rpoplpush 使用 lmove替代

brpoplpush 使用blmove替代

lmove、blmove
